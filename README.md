# Сорванцы — Max Bot

Бот мессенджера [Max](https://max.ru) для магазина детской одежды **Сорванцы**.

## Функционал

- Регистрация покупателей с данными о детях
- Выдача фиксированной скидки после регистрации
- Хранение профиля: имя, телефон, данные детей (имя, пол, дата рождения, класс)
- Поддержка нескольких детей на одного покупателя

## Быстрый старт

```bash
# 1. Установить зависимости
pip install -e .

# 2. Настроить окружение
cp .env.example .env
# Заполнить .env

# 3. Применить миграции БД
python scripts/migrate.py

# 4. Запустить бота
python main.py
```

## Требования

- Python 3.11+
- PostgreSQL 14+
- Доступ к сети Max Messenger

## Структура

```
main.py          — точка входа
config.py        — конфигурация
src/
  handlers/      — обработчики сообщений
  models/        — работа с БД
  services/      — бизнес-логика
  db/            — подключение к БД
docs/
  max-botapi.md  — документация по API библиотеки maxapi
scripts/
  migrate.py     — применение миграций
```
